# ryokochan
LINEインターンシップで作成した旅行プラン作成botです。
# 機能概要
日付、場所、旅行プランなどを選択することで条件にあったプランを提示してくれるline botです。
summer internshipのハッカソンでエンジニア２人で作成したもので、エンジニアリング賞をいただきました。

りょうこちゃん　と呼んで使うことができます。中断したい場合は　ちゅうし　と発言してください
